Outline¶

  • Computing image gradients
  • Constructing histogram of oriented gradients
  • Comparing histograms

Task¶

  1. Given an image $I$, compute $\frac{\partial I}{\partial x}$
  2. Given an image $I$, compute $\frac{\partial I}{\partial y}$
  3. Given an image $I$, compute gradient magnitude and edge orientation for each pixel. Recall that edge orientation refers to the normal to the edge passing through a pixel.
  4. Color each pixel according to its edge orientation.
  5. Construct gradient magnitude histogram for pixels with non-zero gradient magnitude. Use 32 bins.
  6. Construct gradient orientation histogram for pixels with non-zero gradient magnitude. Use 360 bins. We assume that gradient orientations vary between -180 and +180 (in degrees).
  7. For a given image $I$ and its rotated version $I_\mathrm{rot}$, show that their gradient magnitude histogram stays the same, but that their gradient angle histogram changes. Implement a method that returns True if the two histograms are the same, False otherwise.
